# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.1.x   | :white_check_mark: |
| < 1.1   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, please
[create an issue](https://github.com/integreat-io/integreat-authenticator-googleauth/issues)
in this repo.
